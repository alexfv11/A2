*¿Qué es qit?*

Git es un software de control de versiones gratis y de código abierto.Fue creado por Linus Torvalds en 2005. Esta heeerranienta es un sistema de control de versiones que fue inicialmente desarollado para trabajar con varios desarolos en el núcleo de Linux.

Esto significa básicamente que Git es un rastreador de contebido, en particular para almacen código.

Los proyectos de la vida real generalmente tienen múltiples desarrolladores trabajando en paralelo. Así que necesitan un sistema de control de versiones como Git para asegurarse de que no hay conflictos de código entre ellos.

Además, los los requerimentos en este tipo de proyectos cambian constantemente. Así que un sistema de control de versiones permite a los dearolladores revertir y regresar a una vesión anterior de su código.

El sitema de ramas en Git permite a los desarolladores trabajar individualmente en una tarea.

Por ejemplo:

    una rama->una taarea
    una rama -> un desarrollador

>

Básicamente, se puede pensar en Git como una aplicación de software pequeña que controla tu código base, si eres un desarrollador.
